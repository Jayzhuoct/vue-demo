# Vue Demo

这是一个用于学习 Vue.js 的示例项目。

## 项目结构

- `src/`：包含项目的源代码。
- `public/`：包含项目的静态资源。
- `tests/`：包含项目的单元测试。

## 开始使用

1. 克隆项目到本地：

   ```bash
   git clone https://github.com/your-username/vue-demo.git
   ```

2. 安装依赖：

   ```bash
   cd vue-demo
   npm install
   ```

3. 启动开发服务器：

   ```bash
   npm run serve
   ```

4. 在浏览器中访问 http://localhost:8080 查看应用程序。

## 运行测试

运行单元测试：

```bash
npm run test:unit
```
